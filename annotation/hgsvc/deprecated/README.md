## Deprecated file: HGSVC labels

The table in this folder (`quality_labels_high-med-clust.tsv`) contains Strand-seq library quality labels derived on the basis
of the annotation files in the parent folder. These labels were used for training the first set of machine learning models
for ASHLEYS. The updated implementation of the ASHLEYS QC pipeline includes deriving these labels directly from the raw annotation
files (see HGSVC Strand-seq blacklist files and Jupyter Notebooks in the PGAS repository for the original code). Hence, this file
is considered deprecated and should no longer be used.
